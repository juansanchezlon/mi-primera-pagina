### Algoritmo 
Un algoritmo es una secuencia de pasos que se deben ejecutar para resolver un conflicto, de manera sistemática, en un evento único donde las condiciones o restricciones de base de un problema de similar naturaleza sean las mismas, independiente de sus valores. Esto quiere decir que, aunque la cantidad y el tipo de condiciones sean iguales, no deberá importar su valor.

### Lenguajes de programación 
Un lenguaje de programación es un lenguaje formal con reglas gramaticales definidas que permite a un programador escribir un conjunto de instrucciones, para controlar el comportamiento de una computadora; con el objetivo de obtener datos o ejecutar tareas.

Al conjunto de órdenes escritas se le denomina programa


### Tipos de Lenguajes de Programación
Los lenguajes de programación pueden ser clasificados de acuerdo a como son usados por la máquina o sistema operativo:
- Interpretados: las instrucciones son traducidas una a una por una aplicación (intérprete). Ejemplos: Python, Perl, Bash.
- Compilados: las instrucciones son traducidas en grupo por una aplicación (compilador). Ejemplos: Fortran, C/C++.
- Intermedios: en parte compilados, en parte interpretados. Ejemplos: Java, Scala.
Los lenguajes de programación también pueden ser clasificados de acuerdo a su nivel de abstracción:
- Alto nivel: la información compleja es representada con objetos simples. Ejemplos: Python, C++, C#, Java.
- Bajo nivel: la información compleja debe representarse literalmente. Ejemplos: Assembly y Machine.

### Entornos de desarrollo

Están diseñados para que los programadores puedan escribir sus programas con ayuda, encontrar las mejores prácticas o soluciones, y crucialmente, verificar el funcionamiento del código a través de la ejecución de pruebas

### Concepto de algoritmo
Para plantear la estructura básica del algoritmo, podemos identificar tres elementos básicos: entradas, procesos y salidas.
- Entradas: son los insumos requeridos para dar solución al problema, es decir, datos, los cuales pueden presentarse en múltiples formas: números, nombres, fechas, imágenes, archivos de audio, señales de onda, entre otros.
- Procesos: conforman las acciones que se encargan de procesar los datos de entrada con el propósito de devolver los resultados esperados.
- Salidas:  corresponden a la solución del problema. Cuando las salidas no logran el objetivo de dar solución al problema, se puede concluir que el algoritmo estuvo mal diseñado o que se presentaron fallos en alguna parte del proceso.

###  Variables y constantes
Son los elementos que permiten al programa almacenar y manipular datos durante su ejecución.
- Variables: un espacio en la memoria del ordenador que puede cambiar su valor a lo largo de la ejecución del algoritmo. Se usan para datos que se modifican. Ejenplo: un contador, un saldo.
- Constantes: un espacio en la memoria cuyo valor permanece fijo e inalterable durante toda la ejecución del algoritmo. Se usan para valores que no deben cambiar. Por ejemplo:  un impuesto fijo.

### Tipos de datos 
Definen qué clase de información se va a almacenar en una variable o constante, y cómo el ordenador debe interpretarla. Podemos encontrar: 
- Numérico entero
- Numérico decimal
- Cadena de texto
- Booleano 
- Fecha
- Binario 

### Tipos de datos en Python 
Python tiene solo el tipo numérico, aunque se puede adaptar a alguno de los anteriores, dependiendo del contenido del dato que se le asigna.
Otro tipo de dato puede ser:
- Nmeros complejos
- Tupla
- Lista
- Dccionario

### Operadores 

Los operadores son símbolos especiales que se utilizan para realizar operaciones sobre variables y constantes.
Tipos de Operadores:
- Aritméticos: realizan cálculos matemáticos (+, -, *, /, %).
- Relacionales: comparan dos valores, devolviendo un resultado lógico (>, <, >=, <=, ==, !=).
- Lógicos: combinan resultados lógicos o booleanos (AND (y), OR (o), NOT (no)).
- Asignación: asignan un valor a una variable (=, +=, -=).


### Ejemplo de código Python: suma de dos números
# 1. Asignación de variables (Entrada)
numero_a = 5
numero_b = 8

# 2. Proceso (Operación aritmética)
resultado_suma = numero_a + numero_b

# 3. Salida (Mostrar el resultado)
print(resultado_suma)
